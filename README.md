## About the Project
This project is a personal portfolio website designed to highlight my work and present my professional profile. It is structured to be responsive, user-friendly, and visually appealing. The website includes sections for:
- About Me
- My Projects
- Skills
- Contact Information

## Features
- Fully responsive design for desktop, tablet, and mobile
- Interactive navigation
- Clear presentation of projects and skills
- Contact section with email link

## Technologies Used
- **HTML5** – for content structure
- **CSS3** – for styling, layout, and responsive design
- **JavaScript** – for interactive elements and dynamic content
